# **App Name**: RemedyEye

## Core Features:

- Real-Time Dashboard: Display real-time dashboard view of BMC Remedy ticket status.
- KPI Cards: Display key performance indicators (KPIs) such as total active tickets, pending tickets, breached tickets, and tickets closed today.
- Visual Charts: Display ticket distribution by status (pie chart), tickets per support group (bar chart), and ticket resolution trends (line chart).
- Detailed Ticket Table: Show a sortable, searchable table with ticket ID, title, owner, status, priority, SLA deadline, and creation time; exportable to Excel.
- Automatic Refresh: Automatically refresh dashboard data every 60 seconds using API integration.

## Style Guidelines:

- Primary color: A calm blue (#4682B4) reflecting reliability and focus. 
- Background color: Light gray (#F0F8FF), for a clean, uncluttered backdrop that's easy on the eyes in a brightly lit environment.
- Accent color: A gentle yellow (#FFDB58), which draws attention and signals the importance of updates, actions and alerts related to critical KPI and pending service level agreements. 
- Clear, sans-serif font (e.g., Open Sans, Helvetica) for easy readability on screens.
- Simple, consistent icons for ticket status, priority, and actions.
- Clean, grid-based layout for organized and responsive display on TV screens.
- Subtle transitions for data updates to provide a dynamic but non-distracting view.
